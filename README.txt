THE WEDGE GROUP PRESENTER — MODERN PROTOTYPE

To open the prototype, double-click index.html.

Included in this pilot:
- Modern responsive main menu
- The Wedge Workshop objective screen
- Click-to-reveal company differentiation exercise
- Staged hidden-pain interaction with reset behavior
- Four selectable Ladder of Abstraction examples
- Working Home, Back, Next, Calculator, and About controls

This is a self-contained prototype. It does not require Flash, installation,
an internet connection, or a web server.

Open sales-call.html for the animated Sales Call Structure replica.

BUILD RAPPORT — BUTTON 4
- Select Build Rapport.
- Click the fourth small icon (+).
- The original WedgeProcess worksheet opens as a full-page presentation screen.
- Click the worksheet CLOSE button to return to Build Rapport.
- No PDF viewer, toolbar, scrolling, or internet connection is used.
